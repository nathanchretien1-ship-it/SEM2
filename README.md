# iHealth Demo

## Introduction

Projet Android permettant de recevoir des données de capteurs iHealth

## Configuration de Android Studio

Le projet est prévu pour fonctionner sur la configuration suivante :

- JDK : 17

You can configure it with : File > Project Structure... > SDK Location > Gradle Settings > Gradle
JDK

- Android Graddle Plugin Version : 8.2.2
- Gradle Version : 8.6

You can configure it with : File > Project Structure... > Project
