# iHealth Demo

## Introduction

Projet Android permettant de recevoir des données de capteurs iHealth

## Configuration de Android Studio

Le projet est prévu pour fonctionner sur la configuration suivante :

- JDK : 11 You can configure it with : File > Project Structure... > SDK Location > Gradle
  Settings > Gradle JDK
  [Gradle Location](images/Gradle-Location.png)
  [JDK Version](images/Config-JDK.png)

- Android Graddle Plugin Version : 4.0.2
- Gradle Version : 6.1.1 You can configure it with : File > Project Structure... > Project
  [Gradle Version](images/Config-Gradle.png)
